## ðŸ“Š The Analytical Framework
The Altman Z-Score serves as a multivariate predictive model, assessing corporate stability through five critical financial pillars:

* **$X_1$ (Liquidity Index):** Working Capital / Total Assets
* **$X_2$ (Cumulative Profitability):** Retained Earnings / Total Assets
* **$X_3$ (Operating Efficiency):** EBIT / Total Assets
* **$X_4$ (Market Solvency):** Market Cap / Total Liabilities
* **$X_5$ (Asset Turnover):** Sales / Total Assets
